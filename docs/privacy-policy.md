# 개인정보처리방침

최종 업데이트: 2026-07-24

## 1. 소개

Nook("당사", "우리", 또는 "서비스")는 사용자가 온라인 콘텐츠를 저장하고, 정리하고, 나중에 다시 발견할 수 있도록 돕는 개인 아카이브 서비스입니다. Nook는 개인 운영자 Hyerim Han이 단독 운영하는 서비스입니다.

본 개인정보처리방침은 Nook가 어떤 정보를 수집하는지, 그 정보를 어떻게 사용하고 보관하는지, 그리고 사용자가 자신의 정보에 대해 어떤 선택권을 가지는지 설명합니다.

## 2. 수집하는 정보

당사는 서비스 제공에 필요한 범위에서 다음 정보를 수집하거나 저장할 수 있습니다.

### 계정 정보

- 이메일 주소
- 로그인 제공자(Apple 또는 Google)와 연동된 인증 식별자
- 로그인 제공자가 제공하는 이름, 표시 이름 또는 프로필 정보(제공되는 경우)
- 계정 생성, 로그인, 세션 유지에 필요한 인증 정보

### 사용자가 저장한 콘텐츠 정보

- 사용자가 Nook에 저장한 URL
- 저장한 콘텐츠의 메타데이터
  - 제목
  - 썸네일
  - 도메인
  - 설명
- 저장한 콘텐츠에 연결된 카테고리 및 태그
- 저장, 생성, 수정, 열람 시각 등의 기록 정보

Nook는 저장한 링크의 제목, 설명, 썸네일 등 메타데이터를 보완하기 위해 해당 URL 또는 관련 제공자에 네트워크 요청을 보낼 수 있습니다. 이 과정에서 해당 웹사이트나 제공자는 자체 정책에 따라 접속 요청 정보를 처리할 수 있습니다.

### 사용자가 설정하거나 생성한 정보

- 온보딩에서 선택한 관심 카테고리
- 사용자가 직접 추가, 수정, 삭제한 카테고리
- 콘텐츠 보기 방식 등 앱 사용 편의를 위한 설정값

### 서비스 이용 관련 정보

- 핵심 기능 제공을 위해 필요한 기본적인 앱 사용 정보
- 로그인 상태 유지를 위한 인증 세션 정보 및 앱 설정값
- 앱 버전, 세션 식별자 등 서비스 안정성 확인에 필요한 정보

일부 인증 세션 정보와 앱 설정값은 사용자의 기기 내 보안 저장소에 저장될 수 있습니다.

### 알림 관련 정보

푸시 알림 기능(저장 후 오랫동안 열어보지 않은 콘텐츠 알림, 콘텐츠 리마인더 등)을 제공하기 위해 다음 정보를 수집·저장할 수 있습니다.

- 기기의 푸시 알림 토큰(Expo Push Token)
- 기기 플랫폼(iOS 등) 및 기기 이름
- 알림 설정값(알림 사용 여부, 채널별 사용 여부, 시간대)
- 발송한 알림의 제목·내용·관련 콘텐츠 식별자 및 열람 시각 등 발송·열람 기록

알림 기능은 사용자가 기기에서 알림을 허용한 경우에만 동작하며, 사용자는 언제든지 앱 내 알림 설정 또는 기기 설정에서 알림을 끌 수 있습니다.

### 클립보드 정보

사용자의 저장 편의를 위해, 앱을 열 때 기기 클립보드에 링크(URL)가 있는지 기기 내에서만 확인하여 저장을 제안할 수 있습니다. 이 확인은 사용자의 기기에서만 이루어지며, 사용자가 직접 저장을 선택하기 전까지 클립보드 내용을 당사 서버로 전송하거나 저장하지 않습니다.

### 서비스 사용 패턴 정보 (분석 이벤트)

당사는 서비스 품질 측정과 개선을 위해 아래와 같은 사용 패턴 이벤트를 기록할 수 있습니다.

- 앱 실행 (`app_opened`)
- 저장 시도 및 실패 (`save_attempted`, `save_failed`)
- 콘텐츠 열람 (`content_opened`)
- 재발견 카드 노출 (`rediscover_impression`)
- 온보딩 완료 (`onboarding_completed`)

해당 이벤트에는 이벤트 이름, 발생 시각, 앱 버전, 세션 식별자, 관련 콘텐츠 식별자 및 오류 사유와 같은 제한된 속성이 포함될 수 있습니다.

현재 Nook는 광고 식별자(IDFA)를 수집하지 않으며, 외부 광고 SDK나 제3자 분석 SDK를 사용하지 않습니다. 분석 이벤트는 사용자의 계정 식별자와 함께 당사의 데이터베이스에 저장되며 광고 목적의 추적이나 판매에 사용되지 않습니다.

## 3. 정보를 사용하는 목적

당사는 수집한 정보를 다음 목적을 위해 사용합니다.

- 계정 생성 및 관리
- 사용자 인증 및 로그인 세션 유지
- 저장한 콘텐츠의 보관, 분류, 표시
- 저장한 링크의 메타데이터 보완
- 저장한 콘텐츠에 대한 카테고리, 태그, 제목 보정 지원
- 사용자가 저장한 콘텐츠를 다시 찾거나 재발견할 수 있도록 돕는 기능 제공
- 재발견 알림 및 콘텐츠 리마인더 등 푸시 알림 발송
- 사용자 문의 대응 및 지원 제공
- 서비스 안정성, 보안, 악용 방지
- 서비스 사용성 측정 및 품질 개선

## 4. AI 처리

Nook는 저장된 콘텐츠를 분류하고 정리하기 위해 AI 서비스를 사용할 수 있습니다.

이 과정에서 다음과 같은 제한된 정보가 AI 처리에 사용될 수 있습니다.

- URL
- 제목
- 도메인
- 설명
- 사용자가 이미 보유한 카테고리 목록

AI 처리는 태그 생성, 기존 카테고리 중 하나의 추천, 제네릭 제목 보정을 위한 목적으로 사용됩니다. AI는 사용자의 기존 카테고리 목록 안에서만 카테고리를 추천하며, 새 카테고리를 생성하지 않습니다.

현재 Nook는 저장 콘텐츠에 대한 AI 요약 생성 기능을 기본 제공하지 않습니다.

## 5. 제3자 서비스

당사는 Nook 운영을 위해 제3자 서비스 제공자를 사용할 수 있습니다. 이 제공자들은 자신의 역할을 수행하는 데 필요한 범위에서만 사용자 정보를 처리합니다.

현재 주요 제공자는 다음과 같습니다.

- Supabase: 인증, 데이터베이스, 저장소 인프라
- Google 또는 Apple: 로그인 및 신원 인증
- Anthropic: AI 기반 콘텐츠 분류, 태그 생성, 제목 보정

당사가 사용자 정보를 제3자 제공자와 공유하는 경우, 해당 제공자는 자신의 서비스 제공 목적 범위에서 정보를 처리합니다. 당사는 합리적인 범위에서 해당 제공자가 본 개인정보처리방침 및 관련 법령에 부합하는 수준으로 정보를 보호하도록 요구합니다.

Nook는 사용자의 개인정보를 광고 목적으로 판매하지 않습니다.

## 6. 보관 기간

당사는 서비스 제공, 법적 의무 준수, 분쟁 해결, 약관 집행에 필요한 기간 동안 사용자 정보를 보관할 수 있습니다.

계정 정보, 저장한 콘텐츠, 카테고리, 분석 이벤트 및 관련 데이터는 다음 중 하나에 해당할 때까지 보관될 수 있습니다.

- 사용자가 직접 데이터를 삭제하는 경우
- 사용자가 계정 삭제를 요청하는 경우
- 서비스 운영 또는 법적 준수를 위해 더 이상 보관이 필요하지 않은 경우

계정 삭제 후에도 법적 의무, 분쟁 해결, 보안 감사 또는 백업 관리상 필요한 일부 기록은 관련 법령이 허용하는 범위에서 제한적으로 보관될 수 있습니다.

## 7. 삭제 및 사용자 선택권

사용자는 자신의 계정 및 관련 데이터 삭제를 요청할 수 있습니다.

사용자는 앱 내 **Profile → 계정 설정 → 계정 삭제하기** 메뉴를 통해 본인의 계정과 저장된 모든 콘텐츠, 카테고리, 분석 이벤트를 직접 삭제할 수 있습니다. 삭제 후에는 동일한 계정 정보로 다시 가입하더라도 이전 데이터는 복구되지 않습니다.

사용자는 저장한 콘텐츠와 카테고리를 앱 내 기능을 통해 직접 삭제하거나 수정할 수 있습니다.

앱 내 삭제 기능을 이용하기 어려운 경우, 아래 지원 이메일을 통해 계정 및 데이터 삭제를 요청할 수 있습니다.

사용자는 언제든지 서비스 이용을 중단할 수 있습니다.

## 8. 데이터 보안

당사는 사용자 정보를 무단 접근, 공개, 변경, 파기 등으로부터 보호하기 위해 합리적인 기술적·관리적 조치를 취합니다. 여기에는 사용자별 데이터 접근 제한, 인증 세션 보호, 데이터베이스 접근 정책 적용 등이 포함될 수 있습니다.

다만 어떤 전송 방식이나 저장 방식도 절대적으로 안전하다고 보장할 수는 없습니다.

## 9. 아동의 개인정보

Nook는 관련 법령상 부모 또는 보호자 동의 없이 사용할 수 없는 아동(예: 대한민국의 경우 만 14세 미만)을 대상으로 설계된 서비스가 아닙니다. 당사는 관련 법령에 위반되는 방식으로 아동의 개인정보를 고의로 수집하지 않습니다.

## 10. 국가 간 데이터 처리

사용자가 서비스 인프라 또는 제3자 제공자가 위치한 국가 외부에서 Nook를 이용하는 경우, 사용자의 정보는 개인정보 보호 수준이 다른 국가로 이전되거나 그 국가에서 처리될 수 있습니다.

## 11. 개인정보처리방침 변경

당사는 서비스 변경, 법령 변경 또는 운영상 필요에 따라 본 개인정보처리방침을 수정할 수 있습니다. 중요한 변경이 있는 경우, 문서 상단의 날짜를 업데이트하고 필요한 경우 앱 또는 웹페이지를 통해 추가 안내를 제공할 수 있습니다.
